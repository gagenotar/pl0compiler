# PL/0 Compiler

## Overview

This project was done by Gage Notargiacomo, Fall 2023.

Author: Gage Notargiacomo

Language: C

## How to run

To Compile:

```bash
gcc -o pl0compiler pl0compiler.c
```

To Execute:

```bash
./pl0compiler input.txt
```

Where input.txt is an input file containing PL/0 code
